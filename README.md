## Setup

1. Clone the repository
    ```bash
    git clone https://github.com/silvermete0r/codeforces-training-planner.git
    cd codeforces-training-planner
    ```
2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the server:
    ```bash
    python app.py
    ```
